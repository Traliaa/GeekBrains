WiFi-Реле Sonoff Basic

Подробное описание товара

Sonoff Basic является экономически эффективным, решением для умного дома, он разработан и произведен ITEAD. Создать свой умный дом стало просто и легко. Sonoff Basic - это беспроводной выключатель, основанный на Wi-Fi, который можно подключать к приборам различных типов и марок. Функции устройства: - поддержка Wifi - Online статус устройства (включено/выключено) - включение/выключение через Интернет - поддержка 8 таймеров (Включение/Выключение по заданному алгоритму) - Управление неограниченным количеством WiFi реле с одного смартфона - Простая установка, подключение и использование Sonoff Basic является доступным устройством, которое предоставляет пользователям умное управления домом. Это беспроводное Wi-Fi реле, которое может подключаться к широкому спектру устройств. Sonoff передает данные на облачную платформу с помощью маршрутизатора WiFi, что позволяет пользователям удаленно управлять всеми подключенными приборами, через приложение eWeLink. Облачным сервером для Sonoff является глобальный сервер Amazon AWS. Работает online или по расписанию (поддерживает до 8 таймеров). Wifi реле Sonoff сделает все бытовые приборы умными. До тех пор пока мобильный телефон имеет доступ к сети, пользователь может удаленно управлять техникой, включая и выключая её из любого места, в любое время. Также доступна установка расписания для устройств, которое может включать в себя обратный отсчет времени, запланированное включение/выключение, и, таким образом помогает пользователям наслаждаться легкой жизнью. Функциональное русскоязычное приложение Ewelink для IOS и Android позволит вам легко управлять всеми функциями устройства Версию приложения для iOS можно скачать в App Store, а Android-версии - в Google Play.
